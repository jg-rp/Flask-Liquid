## Install

Install Django Liquid using [pip](https://pip.pypa.io/en/stable/getting-started/):

```console
pip install flask-liquid
```

Or [Pipenv](https://pipenv.pypa.io/en/latest/):

```console
pipenv install -u flask-liquid
```
